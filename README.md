# note-app

> Tech Stack: Nodejs, Express, Reactjs, MongoDB
> 
> Project on software to create and manage notes

## Usage


## Install dependencies
- frontend
```
cd frontend
npm i
cd ..
```
- backend
```
cd backend
npm i
```
- copy the .env.example file to the .env file and add the configuration variables

## Run App
- frontend
```
cd frontend
npm start
cd ..
```
- backend
```
cd backend
npm start
```
